# Project Images

Place your project images here with the following naming convention:
- `project1.jpg` - Image for Project 1
- `project2.jpg` - Image for Project 2
- `project3.jpg` - Image for Project 3

## Recommended Specifications
- **Format**: JPG, PNG, or WebP
- **Dimensions**: 1200x600px (2:1 aspect ratio) or similar
- **File size**: Keep under 500KB for optimal loading

## Usage
Images will automatically display on the projects page. If an image is not found, a gradient placeholder will be shown instead.
